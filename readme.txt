By default, WooCommerce uses the currency symbol for the United Arab Emirates Dirham (AED) as "د.إ". This ingenidev plugin changes it to "AED".


== Description ==

By default, WooCommerce uses the currency symbol for the United Arab Emirates Dirham (AED) as "د.إ". This plugin changes it to "AED ".
